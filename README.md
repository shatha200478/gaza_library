# البوابة الموحدة لاستعارة الكتب الجامعية

## وصف المشروع
نظام ويب لإدارة واستعارة الكتب بين الجامعات في قطاع غزة، يتيح للطلاب البحث عن الكتب وطلب استعارتها، كما يوفر لوحة تحكم للمسؤول لإدارة الكتب والمستخدمين والطلبات.

## المميزات

### الطالب
- تسجيل الدخول.
- البحث عن الكتب.
- عرض تفاصيل الكتب.
- إرسال طلب استعارة كتاب.
- متابعة حالة الطلبات.

### المسؤول (Admin)
- إدارة الكتب (إضافة - تعديل - حذف).
- إدارة طلبات الاستعارة.
- إدارة المستخدمين.
- عرض الإحصائيات.

## التقنيات المستخدمة

## خطوات التثبيت

1. انسخ المشروع إلى مجلد:
text
htdocs/

2. أنشئ قاعدة بيانات جديدة باسم:

sql
library_system

3. استورد ملف قاعدة البيانات:

text
database.sql

4. عدل بيانات الاتصال بقاعدة البيانات في ملف:

php
db.php

5. شغل Apache و MySQL من XAMPP.

6. افتح المشروع من خلال:

text
http://localhost/Final_ Project
