# 📚 البوابة الموحدة لاستعارة الكتب الجامعية (gaza_library)

## 📝 وصف المشروع
نظام ويب متكامل لإدارة واستعارة الكتب بين الجامعات في قطاع غزة، يتيح للطلاب البحث عن الكتب الأكاديمية المتوفرة وطلب استعارتها وتسهيل الوصول إليها، كما يوفر لوحة تحكم متكاملة للمسؤول لإدارة الكتب، المستخدمين، والطلبات الإدارية.

---

## ✨ المميزات الرئيسية

### 🎓 قسم الطالب:
* تسجيل الدخول وإنشاء حساب: إمكانية إنشاء حساب جديد وتأكيد الهوية.
* البحث المتقدم: تصفح والبحث عن الكتب المتوفرة في المكتبة.
* تفاصيل الكتب: عرض معلومات الكتاب وإمكانية إرسال طلب استعارة فوراً.
* متابعة الطلبات: متابعة حالة طلبات الاستعارة (مقبول / مرفوض / قيد الانتظار).

### 💼 قسم المسؤول (Admin):
* لوحة التحكم (Dashboard): عرض الإحصائيات العامة للموقع.
* إدارة الكتب: إضافة كتب جديدة، تعديل بياناتها، أو حذفها.
* إدارة طلبات الاستعارة: مراجعة طلبات الطلاب والموافقة عليها أو رفضها.
* إدارة المستخدمين: تصفح بيانات الطلاب المسجلين وإدارتها.
